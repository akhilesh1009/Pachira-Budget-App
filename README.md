# Pachira Budget App

## Overview
Pachira Budget App is a comprehensive Android application designed to help users manage their personal finances effectively. The app allows users to track income and expenses, categorise transactions, set budget limits, and visualise spending patterns, manage multiple wallets, achieve savings goals and earn badges for financial milestones.

## Features

### User Authentication
- Secure login and registration using Firebase Authentication
- Password recovery functionality
- User profile management

### Dashboard
- Overview of financial status
- Income and expense summaries
- Category-based spending analysis with progress bars
- Quick access to recent transactions
- Wallet Specific Overviews

### Transaction Management
- Add income and expense transactions
- Categorise transactions
- Add descriptions and dates and images of receipts
- View transaction history
- Filter transactions by date range and category
- View receipt in image viewer from Transaction details

### Category Management
- Create custom categories for both income and expenses
- Assign colors and icons to categories
- Set budget limits for expense categories
- Edit existing categories
- Default categories provided for quick setup

### Multi-Wallet Support
- Create and manage multiple wallets (Cash, Bank, Credit Card, E-Wallet, Savings)
- Track Balances across different accounts
- Wallet specific transaction filtering
- Custom Wallet Colours
- Real-time balance updates on all wallets

### Budget Tracking & Goals
- Set spending limits for expense categories
- Visual progress bars showing budget usage
- Alerts when approaching or exceeding budget limits
- Create savings goals with targets (New Car, Vacation to Bali etc.)
- Visual progress bar showing savings progress
- Add funds to existing goals
- Celebratory pop up message when user achieves their goal
- Notifications appear to remind the user to add to their goals
- Earn Badges when goals are achieved

### Data Visualisation & Trends
- Interactive charts showing spending patterns
- Monthly income and expense summaries with line and pie charts
- Category-based spending analysis
- Progress tracking against budget limits
- Export PDF for Financial Report

### Notifications and Reminders
- Budget reminders for daily, weekly, biweekly, monthly and yearly goals
- Background notification scheduling
- Goal Achievement Notifications

### Widgets & Quick Access
- Home screen widgets for quick balance overview
- View Net Balance Income and Expenses using Graphs

### Gamification & Achievements
- Badges for financial milestones
  - First Steps: Create and achieve your first savings goal
  - Goal Master: Complete 5 savings goals
  - Savings Legend: Complete 10 savings goals
  - Thousand Club: Save R1000
  - Ten Thousand Club: Save R10,000
  - Hundred Thousand Club: R100,000
  - Quick Saver: Complete a goal within 24 hours
  - Consistent Saver: Add to goals for 7 consecutive days
  - Dedication: Add to goals for 30 consecutive days
  - Big Dreamer: Create a goal over $50,000
  - Perfectionist: Reach exactly your goal amount
- Badge details with descriptions and achievement dates

## Technologies Used

- **UI Components**: Material Design, RecyclerView, BottomSheetDialog
- **Architecture**: Activity-based with Firebase integration

## Setup Instructions

### Prerequisites
- Android Studio Arctic Fox (2020.3.1) or newer
- JDK 11 or newer
- Android SDK 30 or newer
- Firebase account

### Installation
1. Clone the repository
2. Open the project in Android Studio
3. Create a Firebase project and add the `google-services.json` file to the app directory
4. Configure Firebase Authentication and Realtime Database in the Firebase console
5. Build and run the application on an emulator or physical device

## Usage Instructions

### First-time Setup
1. Register a new account or log in with existing credentials
2. The app will automatically create default income and expense categories
3. Navigate to the dashboard to see an overview of your finances

### Adding Transactions
1. From the dashboard, tap the "+" button to add a new transaction
2. Select whether it's an income or expense
3. Choose a category, enter the amount, add a description, and select a date
4. Tap "Save" to record the transaction

### Managing Categories
1. When adding a transaction, tap the "+" button next to categories
2. Enter a name for the category
3. Select a color and icon
4. For expense categories, set a budget limit if desired
5. Tap "Create Category" to save

### Viewing Transactions
1. From the dashboard, tap on the income or expense card
2. Use the date filter to select a specific period
3. View transactions grouped by category with progress bars for expenses
